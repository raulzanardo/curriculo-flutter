## English Version

### About

This is my personal portfolio and interactive resume, designed to showcase my experience as a Flutter developer specializing in industrial automation applications. The project demonstrates my skills in mobile and web development, UI/UX design, and internationalization.

### Features

- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Multi-language Support**: Available in Portuguese (BR) and English (US)
- **Dark/Light Theme**: Toggle between dark and light modes for comfortable viewing
- **Interactive Portfolio**: Detailed case studies of my projects, including the CoPal palletization system
- **Modern UI**: Built with Material Design 3 principles
- **Clean Architecture**: Well-organized codebase following Flutter best practices

### Technologies

**Framework & Language:**
- Flutter 3.10.4+
- Dart 3.10.4+

**Key Packages:**
- `provider` - State management
- `i18n_extension` - Internationalization
- `flutter_svg` - SVG rendering
- `url_launcher` - External links
- `carousel_slider` - Image galleries

**Platform Support:**
- Web (Chrome, Safari, Firefox, Edge)
